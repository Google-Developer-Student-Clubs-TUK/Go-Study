### GO-Study (이태형)

## 초기 설정
- study의 주제에 따라 폴더로 분리
- package 와 폴더 이름은 통일
- package 내 print 파일을 제외한 모든 파일은 private function 으로 구성

## 1. Go By Example
- link: https://gobyexample.com
- package: gobyexample
- exec: gobyexample.PrintExample("파일명")
